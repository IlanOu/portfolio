# Portfolio

## See [here](https://ilanou.github.io/portfolio/) my portfolio !

## To install :

```bash
git clone https://github.com/IlanOu/portfolio.git
```

## To run :

### Install Vite :

```bash
npm install vite
```

```bash
npm run dev
```

## To build :

```bash	
npx vite build
```

## To deploy on github on the [gh-pages](https://github.com/IlanOu/portfolio/tree/gh-pages) branch :

```bash
npx gh-pages -d dist
```
