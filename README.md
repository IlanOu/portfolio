## Quick launch

### Initialization

`pnpm` init

```bash
npm install -g pnpm
```

```bash
pnpm install
```

Sharp

```bash
pnpm add sharp
# or, if not working : 
pnpm add -f sharp
```

Astro

```bash
pnpm install astro
```

Vercel Analytics

```bash
pnpm i @vercel/analytics
```

### To run the project

```bash
npm run dev
```

### To build and preview the project

```bash
npm run build
```

```bash
npm run preview
```
